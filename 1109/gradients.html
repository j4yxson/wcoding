Gradients

What is a gradient?

The gradient CSS data type is a special type of image that consists of a progress transition between two or more colors, also known as color stops


There are 3 types of gradients

  • Linear Gradients
  • Radial Gradients
  • Conic Gradients

Linear Gradients ( linear-gradient() )

These types of gradients are exactly as they sound, they move along in a linear fashion

They can move by default from top to bottom, but you can also specify the direction in your code as well

Examples

Default (Top to Bottom)
background-image: linear-gradient(red, white, blue);
Left to Right
linear-gradient(to right, red, white, blue);
Diagonal
linear-gradient(to bottom right,red, white, blue);

Angle
linear-gradient(270deg, red, white, blue);
Repeating
repeating-linear-gradient(to right, red, white, blue 33%);
Transparency
linear-gradient(to right, rgba(255, 0, 0, 0), rgba(255, 0, 0, 1));

Linear Positioning
linear-gradient(to right, red 20%, orange 20% 40%, yellow 40% 60%, green 60% 80%, blue 80%)

Radial Graidents ( radial-gradient() )

Radial gradients are similar to linear gradients, except that they radiate out from a central point.

You can dictate where that central point is and can also make them circular or elliptical.

Examples

Default Radial
radial-gradient(red, yellow, green)
Spaced Radial
radial-gradient(red 5%, yellow 15%, green)
Circle Radial
radial-gradient(circle, red, yellow, green)

Repeating Radial
repeating-radial-gradient(red, yellow 10%, green 15%);
Radial Center Position
radial-gradient(at 0% 30%, red 10px, yellow 30%, #1e90ff 50%);
Ellipses Closest Side
radial-gradient(ellipse closest-side,red,yellow 10%,#1e90ff 50%,beige);

Ellipses Closest Corner
radial-gradient(ellipse closest-corner, red, yellow 10%, #1e90ff 50%, beige)
Circle Closest Side
radial-gradient(circle closest-side at 25% 75%, red, yellow 10%, #1e90ff 50%, beige);
Circle Closest Corner
radial-gradient(circle closest-corner at 25% 75%, red, yellow 10%, #1e90ff 50%, beige)

Conical Graidents ( conic-gradient() )

A conic gradient is a gradient with color transitions rotated around a center point.

To create a conic gradient you must define at least two colors.

Examples

3 Color Conic Gradient
conic-gradient(red, yellow, blue);
Conic Degrees
conic-gradient(red 45deg, yellow 90deg, green 210deg);
Pie Chart
conic-gradient(red, yellow, green, blue, black); border-radius: 50%;

Conic Gradient with Angle
conic-gradient(from 90deg, red, yellow, blue)
Conic Center Position
conic-gradient(at 60% 45%, red, yellow, blue)
Mac Spinning Beach Ball
conic-gradient(red, yellow, lime, aqua, blue, magenta, red)
