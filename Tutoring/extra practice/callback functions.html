Problem Set 1: Callback Functions
            Write a function double that takes a number and a callback function as arguments, and the callback should be applied to the double of the number.

            Create a function sum that takes two numbers and a callback function. The callback function should be applied to the sum of the two numbers.

            Implement a function getUserData that takes a user ID and a callback function. The callback function should be called with the user data retrieved from an imaginary database.
